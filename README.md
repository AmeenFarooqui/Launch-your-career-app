# Launch Your Career App

A mobile app prototype built with [Proto.io](https://proto.io) for the **Launch Your Career** platform — a career development and job-readiness tool.

## Previewing the Prototype

Open either HTML file directly in your browser (no server needed):

| File | Description |
|------|-------------|
| `preview.html` | Wrapped in an iPhone device skin (recommended) |
| `fullscreen.html` | Fullscreen / borderless view |

## Project Structure

```
├── preview.html          # Device-framed prototype preview
├── fullscreen.html       # Fullscreen prototype preview
├── readme.txt            # Proto.io export notes
└── src/
    ├── index.html        # Core app entry point
    ├── assets/
    │   ├── branding/     # App icons (iOS, Android, splash screens)
    │   ├── generated/    # Auto-generated SVG assets
    │   └── f.../         # Custom SVG/JSON assets
    ├── bodymovin/        # Lottie/Bodymovin animation data
    ├── data/
    │   └── data.js       # App screen and component data
    ├── images/
    │   └── player/       # Player UI assets and device skins
    ├── libraries/        # Proto.io component libraries
    │   ├── common/
    │   ├── generic2/
    │   ├── ios7/
    │   └── ios9/
    ├── scripts/          # Player and plugin scripts
    │   └── plugins/      # jQuery, Lottie, A-Frame
    ├── stylesheets/      # Player and preview styles
    └── webkitmask/       # WebKit mask image utilities
```

## Notes

- Do not modify files inside `src/` — they are auto-generated by Proto.io.
- To make changes to the prototype, edit it in [Proto.io](https://proto.io) and re-export.
