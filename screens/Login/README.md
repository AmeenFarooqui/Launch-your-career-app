# Login Screen

Returning user authentication screen.

## Screen Layout

```
+-----------------------+
| <- Purple Header      |
|        LOGIN          |
|                       |
|  [Email            ]  |
|  [Password         ]  |
|                       |
|       [LOGIN]         |
|                       |
|  Decorative shapes    |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Back arrow | Returns to Start Page |
| Header | Purple rounded rectangle |
| Title | "LOGIN" bold text |
| Email field | Text input |
| Password field | Secure text input |
| LOGIN button | Submits login form |
| Decorative shapes | Accent rectangles |

## Navigation

| Action | Goes to |
|--------|---------|
| Back arrow | Start Page |
| LOGIN button | Home screen |
