# Store Screen

Rewards store where students can browse and redeem prizes.

## Screen Layout

```
+-----------------------+
|  Store                |
|                       |
|  +------+  +------+  |
|  |      |  |      |  |
|  | Item |  | Item |  |
|  +------+  +------+  |
|                       |
|  +------+  +------+  |
|  |      |  |      |  |
|  | Item |  | Item |  |
|  +------+  +------+  |
|                       |
|  (scrollable feed)    |
|                       |
| [Home][?][Top][Me][Store] |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Header | "Store" title |
| Scrolling Feed | Grid of prize/reward items |
| Tab Bar | Bottom navigation |

## Navigation

| Action | Goes to |
|--------|---------|
| Tab Bar | Respective screens |
