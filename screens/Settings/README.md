# Settings Screen

App settings and account management screen.

## Screen Layout

```
+-----------------------+
| <-  Settings          |
|                       |
|  PROFILE              |
|  [Option           >] |
|  [Option           >] |
|                       |
|  Notifications        |
|  [Option      [ON ] ] |
|  [Option      [OFF] ] |
|                       |
|  Support              |
|  [Option           >] |
|  [Option           >] |
|                       |
|      [Sign Out]       |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Back arrow | Returns to Profile |
| Title | "Settings" |
| PROFILE section | Profile-related list items with arrows |
| Notifications section | Two toggle switches (on/off) |
| Support section | Two list items with arrows |
| Sign Out button | Logs user out |

## Navigation

| Action | Goes to |
|--------|---------|
| Back arrow | Profile screen |
| Sign Out | Start Page |
