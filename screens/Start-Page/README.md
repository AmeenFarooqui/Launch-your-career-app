# Start Page

The landing screen of the app. First thing users see when they open Launch Your Career.

## Screen Layout

```
+-----------------------+
|  ████ Purple Header   |
|   LAUNCH              |
|   YOUR                |
|   CAREER              |
|                       |
|       [Logo]          |
|                       |
|    [GET STARTED]      |
|    [LOGIN]            |
|                       |
|  Decorative shapes    |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Header | Purple rounded rectangle at top |
| Title | "LAUNCH YOUR CAREER" bold white text |
| Logo | App logo image centered |
| GET STARTED button | Red rounded button |
| LOGIN button | Yellow rounded button |
| Decorative shapes | 4 green rotated rectangles at bottom |

## Navigation

| Action | Goes to |
|--------|---------|
| GET STARTED | Sign Up screen |
| LOGIN | Login screen |
