# Challenge Screen

The daily quiz/challenge screen where students answer questions to earn points.

## Screen Layout

```
+-----------------------+
|  Challenge #08  10pts |
|  [Timer: 00:14]       |
|                       |
|  Lorem ipsum dolor    |
|  sit amet question?   |
|                       |
|  ( ) Answer option A  |
|  ( ) Answer option B  |
|  ( ) Answer option C  |
|  ( ) Answer option D  |
|                       |
|    [o] [o] [o] [o]    |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Challenge number | "Challenge #08" title |
| Points badge | "10 pts" reward shown |
| Timer | Countdown timer "00:14" |
| Question text | The challenge question |
| 4 Answer options | Selectable answer rectangles |
| Progress dots | 4 circles showing question progress |

## Navigation

| Action | Goes to |
|--------|---------|
| Correct answer | Correct screen |
