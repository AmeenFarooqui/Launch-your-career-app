# Correct Screen

Success feedback screen shown after answering a challenge correctly.

## Screen Layout

```
+-----------------------+
|                       |
|       [CHECK]         |
|      CORRECT!         |
|                       |
|       +10pts          |
|  -------------------  |
|       60 pts!         |
|  00:18 sec x 10pts    |
|                       |
|  [See Leaderboard]    |
|  [Share Result]       |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Animated checkmark | Lottie animation on success |
| CORRECT! banner | Green success message |
| +10pts | Points earned this round |
| Divider line | Visual separator |
| 60 pts! | Running total score |
| Time + points | "00:18 sec x 10pts" stat |
| See Leaderboard button | Navigates to rankings |
| Share Result button | Share icon + share action |

## Navigation

| Action | Goes to |
|--------|---------|
| See Leaderboard | Leaderboard screen |
