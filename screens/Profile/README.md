User profile screen showing stats, rank, streak, and earned prizes.

## Screen Layout

```
+-----------------------+
|  [Person Icon]  [gear]|
|  Kedus Melaku         |
|  Palatine HS          |
|                       |
|  [fire] 12            |
|  Highest Streak       |
|                       |
|  60 pts               |

|                       |
|  Earned PRIZES        |
|  - MacBook            |

```

## Components

| Element | Description |
|---------|-------------|
| Avatar icon | Person outline image |
| Name | "Kedus Melaku" |
| School | "Palatine HS - Palatine" |
| Settings icon | Gear icon top right |
| Streak stat | Fire icon + highest streak count (12) |
| Points | Total points earned (60 pts) |
| Rankings | Highest rank by state, city, school |
| Prizes section | List of earned prizes |
| Tab Bar | Bottom navigation |

## Navigation

| Action | Goes to |
|--------|---------|
| Settings icon | Settings screen |
| Tab Bar | Respective screens |
