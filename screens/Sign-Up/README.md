# Sign Up Screen

New user registration screen where students create their account.

## Screen Layout

```
+-----------------------+
| <- Purple Header      |
|  Create your ACCOUNT  |
|                       |
|  [Full Name        ]  |
|  [Email            ]  |
|  [Password         ]  |
|  [School Name      ]  |
|  [State v] [Grade v]  |
|                       |
|       [SIGN UP]       |
+-----------------------+
```

## Components

| Element | Description |
|---------|-------------|
| Back arrow | Returns to Start Page |
| Header | Purple rounded rectangle |
| Title | "Create your ACCOUNT" text |
| Full Name field | Text input |
| Email field | Text input |
| Password field | Secure text input |
| School Name field | Text input |
| State dropdown | Select state |
| Grade dropdown | Select grade |
| SIGN UP button | Submits registration form |

## Navigation

| Action | Goes to |
|--------|---------|
| Back arrow | Start Page |
| SIGN UP button | Home screen |
